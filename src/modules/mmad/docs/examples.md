# Examples & Use Cases

This section provides practical examples for using MMAD.

---

## Example Workflows

### Example 1: New Digital Product (Full Journey)

A digital entrepreneur launching a new online course:

1. **Brand Brief** — Discover positioning, ICP, and brand voice through guided conversation
2. **Mandala Builder** — Cross funnel stages × creative types to generate 12 production briefs
3. **Ad Batch Generator** — Generate 40 ad variations from the top 4 Mandala briefs
4. **Rapid Test Protocol** — Test with $100 over 3 days, CPC optimization
5. **Performance Review** — After 72h, identify 5 winning ads
6. **Dynamic LP Pipeline** — Each winner gets a dedicated LP with matching headline
7. **Performance Review** — 72h later, review LP conversion rates
8. **Scale** — Double down on the winning ad+LP combination

### Example 2: Scaling Perpetual Campaigns

A power user running ongoing campaigns:

1. VTSD Specialist consults sidecar (campaign history, tested premises)
2. Generates new Mandala focused on untested angles
3. Batch Generation → Rapid Test → Dynamic LP → 72h Review → Scale
4. Continuous loop: each 72h cycle feeds the next

### Example 3: Content Strategy Setup

Building a content machine:

1. **Brand Brief** — Establish brand voice and positioning
2. **Content Plan** — Define 4 editorial pillars, monthly calendar, repurpose strategy
3. **Blog Automation Setup** — Configure Automarticles or custom pipeline
4. **Market Research** — Voice Mining to find ICP's exact language
5. Use findings to refine content pillars and topics

## Common Scenarios

### "I have a product but no marketing"
Start with Brand Brief → Funnel Design → Content Plan. Build the foundation before producing.

### "I need more leads fast"
Ad Batch Generator → Rapid Test → Dynamic LP Pipeline. Speed protocol for quick results.

### "My ads aren't converting"
Performance Review → CRO Specialist → Market Research (Voice Mining). Data-driven optimization.

### "I need a sales page"
VSL Structure (for video) or Landing Page Builder (for text). Both need Brand Brief first.

### "I want to automate my blog"
Content Plan → Blog Automation Setup. Choose Automarticles (easy) or custom pipeline (flexible).

## Tips & Tricks

### Facilitation Over Generation
Strategic workflows ask questions — don't try to rush through them. The Brand Brief is only as good as the thought you put into answering. Let the agent guide you to discoveries.

### The Mandala is Your Creative Engine
The Mandala da Criatividade Infinita is the secret weapon. It systematically generates creative angles you'd never think of alone. Trust the matrix.

### Let the Market Decide
Don't pick your favorite ad. Generate 40, test 20, let CPC data choose the winner. Your intuition is valuable for strategy, not for predicting what converts.

### Zero Cognitive Dissonance
The Dynamic LP Pipeline exists because the #1 conversion killer is mismatch between ad and landing page. Same headline, same promise, same visual language.

### 72h Rhythm
The Performance Analyst operates on a 72h cycle. This is enough data for signal, not so much that you waste budget on losers.

### Sidecar Memory
Three agents (VTSD Specialist, Performance Analyst, Automation Architect) maintain memory across sessions. They learn your patterns and improve over time.

## Troubleshooting

### Common Issues

**"The Brand Brief workflow is too slow"**
It's designed to be thorough. The Brand Brief is the foundation for everything. Rushing it leads to generic marketing. Trust the process.

**"I don't have a VPS for Dynamic LP Pipeline"**
Set `vps_deploy_enabled: false` in config. You can still use Landing Page Builder to create individual LPs and deploy manually.

**"The Rapid Test needs more budget"**
$100 is the minimum viable test. If you have more budget, you can test more variations. The protocol scales — the framework stays the same.

**"Voice Mining isn't finding results"**
Try different platforms (Reddit, Facebook groups, Amazon reviews, Quora). The ICP talks somewhere — you just need to find where.
