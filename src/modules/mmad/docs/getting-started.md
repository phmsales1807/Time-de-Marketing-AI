# Getting Started with MMAD

Welcome to MMAD! This guide will help you get up and running.

---

## What This Module Does

O MMAD (Marketing Method of Agile AI-Driven Development) é um framework completo de marketing digital que trata cada especialidade como um agente de IA com persona, expertise e workflows próprios. Em vez de gerar marketing genérico, ele facilita a descoberta estratégica do seu negócio específico e depois executa em escala com consistência.

---

## Installation

If you haven't installed the module yet:

```bash
bmad install mmad
```

Follow the prompts to configure:
- Project name and type
- Priority platforms (Instagram, YouTube, TikTok)
- Output folders
- VPS availability for Dynamic LP Pipeline

## First Steps

### 1. Create Your Brand Brief

Start with the Brand Strategist agent. This is the foundation — everything else depends on it.

The Brand Brief workflow will guide you through 7 collaborative steps:
1. Problema & Mercado
2. ICP (Ideal Customer Profile)
3. Posicionamento
4. Tom de Voz
5. Promessa & Oferta
6. Narrativa & Storytelling
7. Consolidação

The agent asks 1-2 questions per turn. You are the expert on your business; the agent brings method and process.

### 2. Build Your Mandala

With the Brand Brief ready, invoke the VTSD Specialist to build your Mandala da Criatividade Infinita. This crosses funnel stages with creative types to generate production briefs at scale.

### 3. Generate Ad Variations

Use the Copywriter with the Ad Batch Generator to create 20-40 ad variations from Mandala briefs. The market decides the winner.

### 4. Test with Rapid Test Protocol

$100, 3 days, CPC optimization. The Ads Strategist sets up the test. After 72h, you'll know your winners.

### 5. Deploy Dynamic LPs

Each winning ad gets a dedicated landing page with the same headline — zero cognitive dissonance. Deploy automatically to your VPS.

### 6. Review and Iterate

Every 72h, the Performance Analyst reviews data and provides actionable recommendations. Scale winners, pause losers, iterate.

## Common Use Cases

### New Digital Product (Full Journey)
Brand Brief → Mandala → Batch Ads → Rapid Test → Dynamic LPs → 72h Review → Scale

### Scaling Perpetual Campaigns (Power User)
VTSD Specialist consults history → New Mandala → Batch → Test → LP → Review → Loop

### Content Strategy
Brand Brief → Content Plan → Blog Automation Setup → Platform distribution

### Campaign Launch
Brand Brief → Campaign Builder → Email Sequences → Funnel Design → Launch
